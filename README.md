# Rock, Paper, Scissors (Python)

A simple command-line Rock, Paper, Scissors game written in Python featuring a randomized computer opponent.

## 🚀 Features
* **Case Insensitive Input:** Accepts inputs like "Rock", "ROCK", or "rock" without breaking.
* **Input Validation:** Automatically handles invalid choices safely.
* **Infinite Game Loop:** Play as many rounds as you want without restarting the script.
* **Randomized Computer Opponent:** Uses Python's random module to ensure the computer's moves are completely unpredictable.

## 🛠️ Requirements
* Python 3.x installed on your system.

## 💻 How to Run
1. Run the script in your terminal:
   ```bash
   python game.py
   ```

## 🎮 How to Play
1. Type `rock`, `paper`, or `scissor` when prompted.
2. The computer will instantly make its random choice.
3. The game will announce the winner based on standard rules.
4. Type `y` to play another round or `n` to exit.
